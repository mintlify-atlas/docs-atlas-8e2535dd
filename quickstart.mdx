This guide will help you create your first QR code with Vue QR Style. We'll start with a basic example and then show you how to customize it.

<Note>
  Make sure you've completed the [installation](/installation) before following this guide.
</Note>

## Quick Start

<Steps>
  <Step title="Import the component">
    First, import the `QrCodeGenerator` component in your Vue file:

    ```vue
    <script setup>
    import { QrCodeGenerator } from "vue-qr-style";
    </script>
    ```

    <Info>
      If you registered the component globally, you can skip this import step.
    </Info>
  </Step>

  <Step title="Add the component to your template">
    Use the component with the required `data` prop:

    ```vue
    <template>
      <QrCodeGenerator
        data="https://github.com/NaturalDevCR/vue-qr-style"
        :width="300"
        :height="300"
      />
    </template>
    ```

    The `data` prop accepts any string - URLs, text, JSON, or any content you want to encode.
  </Step>

  <Step title="Customize the appearance">
    Add styling options to make your QR code unique:

    ```vue
    <template>
      <QrCodeGenerator
        data="https://github.com/NaturalDevCR/vue-qr-style"
        :width="400"
        :height="400"
        shape="square"
        :margin="10"
        :dotsOptions="{
          type: 'rounded',
          color: '#3eaf7c'
        }"
        :cornersSquareOptions="{
          type: 'extra-rounded',
          color: '#3eaf7c'
        }"
        :backgroundOptions="{
          color: '#ffffff'
        }"
      />
    </template>
    ```

    <Tip>
      Available dot types: `square`, `dots`, `rounded`, `extra-rounded`, `classy`, `classy-rounded`
    </Tip>
  </Step>

  <Step title="Add a logo (optional)">
    Embed your brand logo in the center of the QR code:

    ```vue
    <template>
      <QrCodeGenerator
        data="https://github.com/NaturalDevCR/vue-qr-style"
        :width="400"
        :height="400"
        image="https://vuejs.org/images/logo.png"
        :imageOptions="{
          hideBackgroundDots: true,
          imageSize: 0.4,
          margin: 0
        }"
        :dotsOptions="{
          type: 'rounded',
          color: '#3eaf7c'
        }"
      />
    </template>
    ```

    <Warning>
      If using external logo URLs, ensure they allow CORS. Local images or same-origin URLs work best.
    </Warning>
  </Step>

  <Step title="Add download functionality">
    Use a component ref to access the download method:

    ```vue
    <script setup>
    import { ref } from "vue";
    import { QrCodeGenerator } from "vue-qr-style";

    const qrRef = ref(null);

    const handleDownload = () => {
      qrRef.value?.download({
        extension: "png",
        name: "my-qr-code"
      });
    };
    </script>

    <template>
      <div>
        <QrCodeGenerator
          ref="qrRef"
          data="https://github.com/NaturalDevCR/vue-qr-style"
          :width="400"
          :height="400"
          :dotsOptions="{ type: 'rounded', color: '#3eaf7c' }"
        />
        <button @click="handleDownload">Download QR Code</button>
      </div>
    </template>
    ```

    Supported formats: `png`, `svg`, `jpg`, `webp`
  </Step>
</Steps>

## Complete Example

Here's a full working example that demonstrates reactive updates:

```vue
<script setup>
import { ref, reactive } from "vue";
import { QrCodeGenerator } from "vue-qr-style";

const qrGenerator = ref(null);

const downloadOptions = reactive({
  name: "my-qr-code",
  extension: "png"
});

const qrOptions = reactive({
  data: "https://github.com/NaturalDevCR/vue-qr-style",
  width: 400,
  height: 400,
  margin: 10,
  shape: "square",
  image: "",
  dotsOptions: {
    type: "rounded",
    color: "#3eaf7c"
  },
  backgroundOptions: {
    color: "#ffffff"
  },
  cornersSquareOptions: {
    type: "extra-rounded",
    color: "#3eaf7c"
  },
  cornersDotOptions: {
    type: "dot",
    color: "#2c3e50"
  }
});

const handleDownload = () => {
  qrGenerator.value?.download({
    extension: downloadOptions.extension,
    name: downloadOptions.name
  });
};

const setVueLogo = () => {
  qrOptions.image = "https://vuejs.org/images/logo.png";
};
</script>

<template>
  <div>
    <QrCodeGenerator
      ref="qrGenerator"
      v-bind="qrOptions"
      :data="qrOptions.data"
    />
    
    <div>
      <input v-model="qrOptions.data" placeholder="Enter URL or text" />
      <select v-model="qrOptions.dotsOptions.type">
        <option value="square">Square</option>
        <option value="dots">Dots</option>
        <option value="rounded">Rounded</option>
        <option value="extra-rounded">Extra Rounded</option>
        <option value="classy">Classy</option>
        <option value="classy-rounded">Classy Rounded</option>
      </select>
      <input v-model="qrOptions.dotsOptions.color" type="color" />
      <button @click="setVueLogo">Add Vue Logo</button>
      <button @click="handleDownload">Download</button>
    </div>
  </div>
</template>
```

<Note>
  The QR code updates automatically when you change any reactive prop value. No manual refresh needed!
</Note>
